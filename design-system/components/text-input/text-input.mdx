# Great Button

`<great-input-text>` is a custom web component that extends HTMLElement it accepts all attributes that you would pass to a `<input>` plus the following custom ones:

-   theme - one of the predefined themes `primary`, `secondary` and `tertiary`

NOTE: Passing custom CSS class or inline style won't take effect as the button comes pre styled, to change the button appearance please pass `theme` attribute
