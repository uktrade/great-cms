Breadcrumbs

Like page titles, breadcrumbs labels should use sentence case - capitalise only the first word except when the breadcrumb contains proper nouns or acronyms e.g. 'Contact us about your investment', 'Market guide for Germany', 'Find out more about DIT'.

Usage in templates

Two levels

Specify the current page's label as the first argument of the tag, then include a single link in the body of the tag:

{% code 'django' %} {% breadcrumbs 'Current page' %} Home {% endbreadcrumbs %} {% endcode %}

Output

{% breadcrumbs 'Current page' %} Home {% endbreadcrumbs %}

Context variables can also be used as the first argument and inside the tag:

{% code 'django' %} {% breadcrumbs page.title %} {{ home_label }} {% endbreadcrumbs %} {% endcode %}

Output

{% breadcrumbs page.title %} {{ home_label }} {% endbreadcrumbs %}

Three levels

Specify the Current page's label as the first argument of the tag, then include two links in the body of the tag:

{% code 'django' %} {% breadcrumbs 'Current page' %} Home Middle label {% endbreadcrumbs %} {% endcode %}

Output

{% breadcrumbs 'Current page' %} Home Middle label {% endbreadcrumbs %}

More levels

Similar to above, but include a number of links in the body of the tag.

{% code 'django' %} {% breadcrumbs 'Current page' %} Level 1 Level 2 Level 3 Level 4 {% endbreadcrumbs %} {% endcode %}

Output

{% breadcrumbs 'Current page' %} Level 1 Level 2 Level 3 Level 4 {% endbreadcrumbs %}
